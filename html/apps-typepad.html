Adding the All for Good Gadget to your TypePad blog

Here's how to set up an All for Good Gadget for a basic blog template on TypePad. You'll be setting up a gadget that will be placed in the narrow right sidebar of a basic three-column blog layout. Start by going to this page.

Under Display settings, the Title is text that will appear above your gadget once it's placed in your blog template. You might enter "All for Good - Pets" and then set pets as a preferred keyword in the Gadget settings. Set the width of your gadget to 160 pixels and height to 380 pixels. In this example, we left the border turned off because the blog column is already very narrow. Set Stories to read to 4. Enter your location.

Click the Get the Code button and a text field will appear below containing the HTML code you need. Click in the text field, select all and copy.

Click in the text field, select all and copy the HTML in the clipboard.

Log into TypePad and on the home page, select the web log into which you wish to add the Gadget, in this case one named "Mike's Blog" under the QuickStart Weblogs tab.

Next, click on the Design tab.

For the Current Design, click on the third section called Content to add or remove sidebar modules.

You will see three columns of scrolling lists of template functions near the top of the page. Under the first column, Categories, select widgets. Then under the second column, Modules, select Embed your own HTML.

Under the third column, Details, click the Add this module button.

A pop-up window will appear. Label the Custom HTML a name that makes sense to you. The label will appear in the design panel for reference only. Then in the HTML field, paste the HTML you saved into the clipboard from the All for Good gadget page. Click save.

A module will appear at the top of the left column with the given label "All for Good Gadget."

Click and drag the module over to the top of the right column.

Click Save Changes.

You'll be switched back to the Current Design options page with a blue bar confirming that the changes to the weblog's design have been saved.

Click the blue View Weblog button at upper right.

And you should see your shiny new All for Good Gadget nested happily in the top of the right column of your blog!
