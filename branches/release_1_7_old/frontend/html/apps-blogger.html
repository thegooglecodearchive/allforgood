Adding the All for Good Gadget to your Blogger blog

Here's how to set up an All for Good Gadget for a basic blog template on Blogger. Start by going to this page.

Under Display settings, the Title is text that will appear above your gadget once it's placed in your blog template. You might enter "All for Good - Pets" and then set pets as a preferred keyword in the Gadget settings. Set the width of your gadget to 200 pixels. In this example, we left the border turned off because the blog column is already very narrow. Set Stories to read to 4. Enter your zip code.

Click the Get the Code button and a text field will appear below containing the HTML code you need. Click in the text field, select all and copy.

Click in the text field, select all and copy the HTML in the clipboard.

Log into Blogger and go to your blog's administration area. Click on the Layout tab. Click the Add a Gadget link on the right.

In the pop up window, click on HTML/JavaScript

Under Title, enter a title for the Gadget that makes sense for your blog. Paste the code you copied earlier under Content. Click Save.

You'll now see the gadget name listed in your sidebar.

Click to view your blog, and you'll see the gadget!
