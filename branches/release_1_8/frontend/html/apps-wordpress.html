Adding the All for Good Gadget to your Wordpress blog

Here's how to set add an All for Good gadget to your WordPress blog sidebar. Note that you can only install the gadget in the sidebar on your hosted blog, not on free accounts with WordPress.com. Start by going to this page.

Under Display settings, the Title is text that will appear above your gadget once it's placed in your blog template. You might enter "All for Good - Pets" and then set pets as a preferred keyword in the Gadget settings. Set the width of your gadget to 180 pixels. In this example, we left the border turned off because the blog column is already very narrow. Set Stories to read to 4. Enter your zip code.

Click the Get the Code button and a text field will appear below containing the HTML code you need. Click in the text field, select all and copy.

Click in the text field, select all and copy the HTML in the clipboard.

Log into WordPress and go to your blog's administration area. Click on the Appearance link on the bottom left. Then click Widgets in the menu that drops down.

Look for the widget called Text and drag it to the sidebar area on the right.

Release the box when it's dragged into the sidebar area. You'll see a popup for you to fill in the details.

Enter a title for the gadget and paste the code you copied earlier into the second box. Click Save on the bottom right.

You'll now see the gadget listed in the sidebar area. Click Visit Site on the top of the page.

That's it! The gadget should now appear on your sidebar.
